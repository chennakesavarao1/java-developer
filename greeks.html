Core HTML Elements

HTML (HyperText Markup Language) is the foundation of web development. Here are some essential elements:

  1. Heading Tags - Define headings from <h1> to <h6>
  2. Paragraph Tags - Structure content with <p>
  3. List Tags - Create ordered (<ol>) and unordered (<ul>) lists
  4. Formatting Tags - <b> for bold, <i> for italic, and <u> for underline
